## Getting Started

To install it, you will need:

* Java 11+
* Maven/Gradle
* Some Lib

## Installing

### Gradle dependency

```Groovy
dependencies {
   implementation group: 'io.github.amayaframework', name: 'amaya-core', version: 'LATEST'
}
```

### Maven dependency

```
<dependency>
    <groupId>io.github.amayaframework</groupId>
    <artifactId>amaya-core</artifactId>
    <version>LATEST</version>
</dependency>
```
